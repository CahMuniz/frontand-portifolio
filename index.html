Quem sou eu

Sou formada em Análise e Desenvolvimento de Sistemas pela Newton Paiva. Durante a graduação, fui monitora de informática em estágio, onde ganhei experiência prática com manutenção de máquinas, montagem, limpeza e configuração de equipamentos — além de apoiar colegas e a comunidade no primeiro contato com a tecnologia.

Hoje, meu foco é desenvolvimento back-end em Java com Spring Boot e bancos de dados relacionais (MySQL), sem deixar de lado boas práticas de front-end, dados e segurança. Tenho disponibilidade e, sobretudo, vontade genuína de ajudar a comunidade a ter acesso a conhecimento e técnicas de programação — esse é o motivo deste site existir do jeito que ele é.

Como um sistema funciona

De forma simples: o código-fonte escrito por uma pessoa desenvolvedora é traduzido (compilado ou interpretado) para instruções de máquina, que são executadas em sequência, lidas entradas e gerando saídas.

Arquitetura & API

O que é uma API

API (Interface de Programação de Aplicações) é um "contrato" que permite que dois sistemas diferentes troquem informações de forma padronizada, sem que um precise conhecer os detalhes internos do outro.

Ética & privacidade

Os comentários são salvos localmente no seu navegador (localStorage) por padrão. Nenhum dado é vendido ou compartilhado com terceiros. Evite publicar informações pessoais sensíveis.

Segurança

O back-end usa parâmetros preparados (JPA/Hibernate) contra SQL Injection, validação de entradas e CORS configurado — seguindo boas práticas como as recomendadas pela OWASP.

Acessibilidade

HTML semântico, textos alternativos, navegação por teclado, contraste adequado e respeito à preferência de "movimento reduzido" do seu sistema operacional.
